# C++

A fully working project with build system.

https://github.com/senior7515/smf-getting-started-cpp



```sh

# 1 clone
git clone --recursive https://github.com/senior7515/smf-getting-started-cpp

# 2 go to project
cd smf-getting-started-cpp

# 3 build
./build.sh

```
